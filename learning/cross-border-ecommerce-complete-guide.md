## 📦 第二部分：选品技能详解

### 2.1 选品核心原则

**✅ 好产品的 7 个标准**

1. **刚需高频** - 用户需要且会重复购买
2. **轻小件** - 重量<500g，体积小，物流成本低
3. **高毛利** - 售价≥成本×3，毛利>60%
4. **无品牌垄断** - 避开大品牌，选择白牌市场
5. **不易碎** - 降低运输损耗
6. **无合规风险** - 避开敏感品类（液体、粉末、电池需特殊渠道）
7. **有差异化空间** - 可以改进包装、功能、组合

### 2.2 选品渠道

| 渠道 | 用途 | 技巧 |
|------|------|------|
| 1688 | 找供应商、看成本 | 按销量排序、看复购率 |
| 速卖通 | 看海外热销 | 按订单量排序、看评价 |
| Amazon | 看价格锚点 | Best Sellers 榜单 |
| TikTok | 看趋势 | 热门标签、爆款视频 |
| Facebook Ad Library | 看竞品广告 | 搜索关键词、看投放时长 |
| Google Trends | 看趋势 | 季节性、地区热度 |

```

─────────────────────────────────────

风险评估:
  - 合规：含电池，需特殊物流
  - 竞争：中等，有差异化空间
  - 季节：全年稳定
  
结论：✅ 推荐选品（毛利高、需求稳）
```

| Google | 搜索意图强 | $10/天 | 3.0-6.0 | 14-30 天 |

### 3.2 广告素材制作要点

**🎬 视频素材 (15-30 秒)**
- 前 3 秒必须抓住注意力（痛点/反差/好奇）
- 展示产品使用场景
- 突出核心卖点（不超过 3 个）
- 结尾明确 CTA（立即购买/了解更多）

**🖼️ 图片素材**
- 高清、无水印
- 展示产品细节
- 使用场景图 + 产品图组合
- 文字不超过 20% 面积

### 3.3 广告账户结构

```
广告账户
├── 广告系列 1: 冷启动测试
│   ├── 广告组 1: 兴趣定向 A (25-45 岁女性)
│   │   ├── 素材 1 (视频)
│   │   └── 素材 2 (图片)
│   └── 广告组 2: 兴趣定向 B (25-45 岁男性)
│       ├── 素材 1 (视频)
│       └── 素材 2 (图片)
├── 广告系列 2: 再营销
│   └── 广告组 1: 加购未购买 (7 天)
└── 广告系列 3: 爆款放量
    └── 广告组 1: Lookalike 1% (购买用户)
```

### 3.4 投放优化策略

**第一阶段：测试期 (Day 1-7)**
- 预算：$20-50/天
- 目标：找到 winning creative + winning audience
- 动作：每 24 小时关停 ROAS<1.5 的广告组

**第二阶段：优化期 (Day 8-14)**
- 预算：$50-100/天
- 目标：放大表现好的广告组
- 动作：每 48 小时增加 20% 预算

**第三阶段：放量期 (Day 15+)**
- 预算：$100+/天
- 目标：稳定 ROAS>3.0，持续放量
- 动作：创建 Lookalike 受众，拓展新素材

## 🛒 第四部分：运营转化技能

### 4.1 落地页优化

**高转化落地页 7 要素**

1. **标题** - 突出核心利益点（省钱/省时/变美/变强）
2. **痛点** - 描述用户当前困扰
3. **解决方案** - 产品如何解决痛点
4. **社会证明** - 用户评价、销量数据
5. **稀缺性** - 限时优惠、库存紧张
6. **风险逆转** - 7 天无理由、质保承诺
7. **CTA** - 明确的购买按钮

### 4.2 定价策略

| 策略 | 适用场景 | 示例 |
|------|----------|------|
| 锚定定价 | 有竞品参考 | 竞品$59，我们$49 |
| 心理定价 | 冲动消费 | $39.99 vs $40 |
| 捆绑定价 | 提高客单价 | 买 2 件 8 折 |
| 阶梯定价 | 多规格产品 | 基础版$29/Pro 版$49 |

### 4.3 转化率优化 (CRO)

**基准数据**
- 电商平均转化率：2-3%
- 优秀水平：4-6%
- 顶级水平：8%+

**优化动作**
- A/B 测试标题、图片、价格
- 优化加载速度（<3 秒）
- 简化结账流程（减少步骤）
- 添加信任标识（安全支付、质保）
- 实时客服（提高咨询转化）

| 海外仓 | 2-5 天 | ¥30/kg+ 仓储费 | 爆款、稳定销量 |

### 6.2 仓储管理

**安全库存公式**
```
安全库存 = (最大日销量 × 最大补货天数) - (平均日销量 × 平均补货天数)

示例:
- 最大日销量：50 单
- 平均日销量：30 单
- 最大补货天数：15 天
- 平均补货天数：10 天

安全库存 = (50 × 15) - (30 × 10) = 750 - 300 = 450 件
```

**补货提醒**
- 当库存 < 安全库存 + (平均日销量 × 补货天数) 时触发
- 提前 15 天联系供应商备货

## 💰 第七部分：财务与风控

### 7.1 成本结构

```
售价 $39.99 (¥280)
├── 采购成本：¥45 (16%)
├── 物流成本：¥25 (9%)
├── 平台佣金：¥42 (15%)
├── 广告成本：¥56 (20%)
├── 支付手续费：¥8 (3%)
├── 退货损耗：¥14 (5%)
└── 净利润：¥90 (32%)
```

### 7.2 现金流管理

**健康指标**
- 现金流天数 > 30 天
- 广告花费占比 < 25%
- 库存周转 < 45 天
- 退货率 < 5%

**风险预警**
- 🔴 现金流 < 15 天：立即减少广告、清理库存
- 🟡 现金流 15-30 天：控制广告预算、加快回款
- 🟢 现金流 > 30 天：健康，可适度扩张

### 7.3 关键财务公式

```
毛利率 = (售价 - 采购成本 - 物流成本) / 售价
净利率 = (售价 - 所有成本) / 售价
ROAS = 广告带来的收入 / 广告花费
ROI = (收入 - 成本) / 成本
客单价 = 总收入 / 订单数
复购率 = 复购用户数 / 总用户数
```

### 8.2 数据驱动决策

**问题诊断流程**
```
GMV 下降
├── 是流量下降吗？
│   ├── 是 → 检查广告花费、素材疲劳、受众饱和
│   └── 否 → 继续
├── 是转化率下降吗？
│   ├── 是 → 检查落地页、价格、竞品
│   └── 否 → 继续
├── 是客单价下降吗？
│   ├── 是 → 检查促销策略、捆绑销售
│   └── 否 → 检查数据统计是否有误
```
