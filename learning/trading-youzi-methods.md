## 一、什么是游资

游资（Hot Money）= 短线投机资金，特点：
- **快进快出**：持股时间短（1-5 天为主）
- **追逐热点**：哪里有赚钱效应就去哪里
- **善于造势**：通过涨停板吸引跟风资金
- **风格鲜明**：不同游资有不同的操作风格

## 三、游资常见操盘手法

### 1. 涨停板吸筹
- 通过快速拉涨停，吸引市场关注
- 封板期间吸纳筹码
- 利用涨停的信号效应带动跟风盘

### 2. 打板接力
- 在个股首板后，次日继续打板接力
- 通过连续涨停制造 "妖股" 效应
- 每个涨停都吸引更多资金关注

### 3. 低吸反包
- 在个股大跌/核按钮后低吸
- 等待市场情绪修复后反弹
- 关键：判断是 "错杀" 还是 "真弱"

### 4. 半路截击
- 在个股涨 3-7% 时介入
- 博取封涨停后的溢价
- 比打板成本低，但确定性也低

### 5. 卡位战法
- 当龙头出现分歧时，买入同题材的补涨股
- 补涨股可能成为新龙头（卡位成功）
- 风险：龙头如果修复，卡位股可能被抛弃

### 6. 集合竞价博弈
- 通过集合竞价的高开幅度判断市场态度
- 竞价抢筹 vs 竞价出货
- 关注竞价量：量大 = 分歧大，量小 = 一致

## 四、游资的龙头战法

### 龙头的定义
- 板块内涨幅最大 / 连板最高的个股
- 市场辨识度最高（最多人讨论）
- 资金体量最大（成交额最高）
- 对板块有带动效应（龙头涨，跟风涨）

### 龙头诞生的过程
```
题材出现 → 多只个股涨停 → 竞争龙头 → 一只脱颖而出 
→ 连板确认龙头地位 → 资金集中涌入 → 加速上涨
```

### 龙头的生命周期
1. **启动期**：首板/二板，市场还不确定谁是龙头
2. **确认期**：三板/四板，辨识度建立，资金认可
3. **加速期**：五板+，一致性最强，量价齐升
4. **分歧期**：开始出现大面/炸板，多空博弈
5. **衰退期**：连续下跌，资金撤离

### 参与龙头的原则
- **买在确认期**（2-3 板最佳，辨识度+安全边际）
- **加仓在加速初期**（4 板确认后小幅加仓）
- **分歧期减仓**（不确定就先走一部分）
- **衰退期不碰**（下跌途中不抄底龙头）

## 五、游资的情绪判断

游资最核心的能力 = **判断市场情绪**

### 情绪判断维度
1. **涨停数量**：多 = 情绪好，少 = 情绪差
2. **炸板率**：低 = 一致看多，高 = 分歧大
3. **连板高度**：高 = 赚钱效应强
4. **首板溢价**：正 = 有接力意愿，负 = 恐惧
5. **跌停数量**：多 = 亏钱效应强
6. **龙头表现**：强 = 市场有灵魂，弱 = 群龙无首

### 这与宝哥的情绪周期完全吻合！
游资的操盘手法本质上就是在情绪周期的不同阶段采取不同策略：
- 冰点 → 潜伏/低吸（逆向思维强的游资）
- 回暖 → 首板试错
- 主升 → 打板接力（最赚钱的阶段）
- 分歧 → 卡位/低吸反包
- 退潮 → 撤退/空仓

## 六、游资的风控

### 止损原则
- **打板止损**：次日低开超过 -3% 考虑止损
- **低吸止损**：跌破买入价 -5% 止损
- **时间止损**：买入后 2 天不涨则考虑卖出
- **逻辑止损**：买入逻辑被证伪则无条件走

### 仓位管理
- 单只个股不超过总仓位的 30%
- 总仓位跟随情绪周期调整
- 浮盈加仓，浮亏不加仓

### 纪律性
- 游资最重要的素质 = **纪律**
- 该走就走，不犹豫
- 该干就干，不恐惧
- 做错认错，不扛单

## 八、与宝哥体系的结合思考

宝哥的 "情绪周期交易体系" 本质上就是**将游资的操盘思维系统化**：

1. **周期定位**（先判断当前情绪阶段）→ 决定仓位和策略
2. **选股**（在正确的阶段选择正确的标的）→ 龙头 / 补涨 / 首板
3. **执行**（打板 / 低吸 / 接力）→ 具体的买卖动作
4. **风控**（止损 / 止盈 / 仓位管理）→ 保护利润

**层级关系：**
```
情绪周期（战略层）
  ↓
选股方向（战术层）
  ↓
买卖操作（执行层）
  ↓
风险控制（保障层）
```
