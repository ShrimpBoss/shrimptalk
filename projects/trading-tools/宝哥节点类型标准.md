# 📋 宝哥节点类型标准

**记录时间**: 2026-03-19  
**来源**: 宝哥口述  
**用途**: 严格按照此标准标记节点类型

---

## 关键节点类型（11 种）

### 1. 冰点
- 市场情绪极度低迷
- 涨停家数极少
- 连板高度被压制

### 2. 冰点修复
- 冰点后的第一个修复日
- 情绪开始回暖
- 龙头先手点

### 3. 龙头断板
- 市场龙头股断板
- 可能引发情绪切换
- 需要观察新方向

### 4. 大分歧修复
- 大分歧后的修复
- 题材开始重新凝聚
- 穿越反包低吸点

### 5. 公告停牌
- 龙头股停牌
- 可能引发切换
- 观察新题材

### 6. 连板切换
- 老周期退潮
- 新周期连板出现
- 高低切换

### 7. 大题材见顶
- 主线题材高潮后见顶
- 后排加速卖点
- 准备切换

### 8. 指数新高
- 指数突破 N 年新高
- 情绪与指数共振
- 新阶段信号

### 9. 9 板切换
- 高度板达到 9 板
- 一定有大行情
- 关键节点

### 10. 强分歧分离
- 强分歧后的分离确认
- 前排 pk 打板
- 唯一前排低吸点

### 11. 节点（通用）
- 其他关键日期
- 需要进一步观察
- 待明确类型

---

## 节点类型叠加

**有的节点是多类型叠加**，比如：
- 冰点 + 龙头断板
- 指数新高 + 9 板切换
- 大分歧修复 + 连板切换

**标记方法**: 用"+"连接，如 `"冰点 + 龙头断板"`

---

## 节点周期框架

```
退潮 → 冰点 → 冰点修复 → 情绪回暖 → 情绪加速 → 高潮/弱分歧 
→ 第一次大分歧 → 分歧修复 → 第二次分歧 → 分歧修复 → 退潮
```

---

## 重要节点特征

### 9 板大节点
- 一定有大行情
- 情绪与指数共振
- 题材大爆发

### 三冰点节点
- 连续三次冰点
- 情绪极致低迷
- 转折信号

### 大题材见顶节点
- 主线题材高潮
- 后排加速
- 龙头断板/跌停

### 指数新高节点
- 指数突破 N 年新高
- 新阶段信号
- 情绪共振

## 操作策略对应

| 节点类型 | 操作策略 |
|----------|----------|
| 冰点 | 观察，等待修复 |
| 冰点修复 | 龙头先手，题材观察 |
| 龙头断板 | 观察切换方向 |
| 大分歧修复 | 穿越反包低吸，节点打板 |
| 公告停牌 | 观察新题材 |
| 连板切换 | 低位切换先手 |
| 大题材见顶 | 后排卖点，准备切换 |
| 指数新高 | 上仓位，怼龙头 |
| 9 板切换 | 大行情，主线补涨 |
| 强分歧分离 | 前排 pk 打板，唯一前排低吸 |
