- A 股相关：中海油服、海油工程、中曼石油、准油股份

### 2. 黄金股超跌反弹 (⭐⭐⭐⭐)

- 赤峰黄金停牌 (实控人变更) 可能带动板块关注度
- A 股相关：山东黄金、中金黄金、银泰黄金、赤峰黄金 (停牌)
**风险**: 美联储鹰派立场未变，反弹空间有限

### 3. 化工/煤化工 (⭐⭐⭐⭐)
**逻辑**:

- 关注煤制甲醇、煤化工龙头企业
- A 股相关：宝丰能源、华鲁恒升、鲁西化工

### 4. 半导体国产替代 (⭐⭐⭐⭐)
**逻辑**:
- 先锋子公司 300 吨光刻胶项目投产，国产替代再下一城
- 美日联合声明涉及关键矿物合作，供应链自主可控重要性提升

- A 股相关：南大光电、晶瑞电材、上海新阳、容大感光

### 5. 物流/航空 (⭐⭐⭐)
