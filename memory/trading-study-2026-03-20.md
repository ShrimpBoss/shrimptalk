# 📚 2026-03-20 交易体系学习 — 周期框架深化

**学习时间**: 2026-03-20 20:00  
**学习主题**: 主升周期 vs 补涨周期、主升节点 vs 切换节点

---

## 💡 核心框架升级

### 原框架：五阶段情绪周期
```
冰点 → 回暖 → 主升 → 分歧 → 退潮
```

### 新框架：两周期 + 两节点
```
主升周期 → 补涨周期
    ↓          ↓
主升节点    切换节点
```

---

## 📊 主升周期 vs 补涨周期

### 主升周期特征（5 个）
| 特征 | 表现 |
|------|------|
| 1. 连板高度 | 不断突破（3 板→4 板→5 板→6 板→7 板） |
| 2. 主线 | 单一核心主线，资金聚焦 |
| 3. 梯队 | 完整（从连板龙头到首板补涨） |
| 4. 炸板率 | <25%（资金一致性强） |
| 5. 成交量 | 持续放大（2 万亿→2.5 万亿→3 万亿） |

### 补涨周期特征（5 个）
| 特征 | 表现 |
|------|------|
| 1. 连板高度 | 受限（5-6 板天花板，无法突破） |
| 2. 主线 | 分化（2-3 个主线，资金分散） |
| 3. 梯队 | 中位股断层（3 板、4 板空缺） |
| 4. 炸板率 | 35%-45%（资金分歧加大） |
| 5. 成交量 | 震荡（场外资金观望） |

## 🎯 主升节点 vs 切换节点

### 主升节点判断标准（5 个信号）

| 1. 连板高度突破 | 5 板→6 板→7 板 |

| 3. 旧主线退潮 | 主线龙头出现大面股（如协创数据 -20%） |
| 4. 新题材试盘 | 有新题材但持续性不确定 |
| 5. 成交量萎缩 | 2.3 万亿→2 万亿→1.8 万亿 |

**操作**: 5 个信号出现 3 个以上 → **减仓到 3-5 成**

## 🎯 操作策略

### 仓位控制
| 周期阶段 | 仓位 |
|----------|------|
| 主升周期 | 7-10 成 |
| 补涨周期 | 5-7 成 |
| 切换节点确认后 | 3-5 成 |
| 冰点期 | 0-1 成 |

### 下周策略

#### 周初（周一、周二）
- **观察炸板率**: 如果>45%，减仓到 3-5 成
- **观察连板高度**: 如果突破 6 板，加仓到 7 成
- **观察储能持续性**: 如果继续强，可聚焦储能

#### 周中（周三、周四）
- **确认周期定位**: 
  - 如果切换节点确认 → 保持 3-5 成
  - 如果补涨周期延续 → 保持 5-7 成
- **聚焦主线**: 去弱留强

#### 周末（周五）
- **防备周末消息面**
- **如果全周炸板率高**: 降到 5 成以下
- **等待下周新周期**

### 核心要点
> **补涨周期后期的核心是【防守】，不是【进攻】。**
> **等切换节点确认后再进攻。**

## 🧠 核心收获

### 1. 周期框架升级
- **原框架**: 五阶段（冰点→回暖→主升→分歧→退潮）
- **新框架**: 两周期 + 两节点（更实用）

### 2. 量化指标完善
- **炸板率标准**: <25% 健康 / 25%-35% 正常分歧 / 35%-45% 退潮 / >45% 冰点
- **连板高度**: 主升期 7 板 + / 补涨期 5-6 板 / 冰点期 2-3 板
- **5 个信号确认法**: 5 个信号出现 3 个以上确认

### 3. 周期 + 节点组合策略
- **补涨周期后期 + 接近切换节点** = 5-7 成仓位 + 防守为主
- **主升周期 + 主升节点确认** = 7-10 成仓位 + 进攻为主
- **切换节点确认后** = 3-5 成仓位 + 等待新周期
