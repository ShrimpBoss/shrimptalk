### 5. 数据准确性教训 ⚠️
**错误案例**：把长城科技（首板，开板 5 次）误判为 2 连板
**根因**：把"开板次数"当成"连板数"
**改进**：
- 脚本直接解析原始数据，不人工判断
- 输出原始数据 + 结构化数据两份，方便核对
- 连板数必须和"X 连板"标注一致

**数据字段的重要性**：
| 字段 | 意义 |
|------|------|
| 价格 | 判断股票位置（低价股/高价股） |
| 换手率 | 判断涨停质量（缩量/放量） |
| 流通市值 | 判断拉升难度（小盘/大盘） |
| 首次封板时间 | 判断强度（早盘板/午后板） |
| 最后封板时间 | 判断是否炸板回封 |
| 开板次数 | 判断分歧程度（一字板/烂板） |
