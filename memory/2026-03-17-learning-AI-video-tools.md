### 3. AI 视频创作工作流 ✅

**完整创作流程**：
```
创意构思 → 文字/图片输入 → AI 生成 → 编辑优化 → 导出发布
    ↓           ↓            ↓         ↓         ↓
  剧本      Prompt/素材    Gen-4.5   Aleph    多平台
```

**最佳实践**：
1. **Prompt 工程**：详细描述场景、光线、镜头运动、情绪
2. **迭代生成**：先生成短片段测试，满意后再延长
3. **多工具组合**：Runway 生成 + 传统编辑软件后期
4. **积分管理**：合理规划积分使用，避免浪费

**适用场景**：
- 🎬 短视频内容创作（TikTok/Reels/Shorts）
- 📺 广告素材快速生成
- 🎭 故事板/概念可视化
- 🏢 企业宣传视频
- 🎮 游戏过场动画
- 📱 自媒体内容批量生产

## 🎯 可立即应用的行动

### 行动项：创建第一个 AI 视频作品

**步骤**：
1. **注册 Runway 免费账户** → 获得 125 积分试用
2. **准备 3 个视频创意** → 每个用 1-2 句话描述
3. **使用 Gen-4 Turbo 测试** → 每个创意生成 5 秒视频
4. **对比效果** → 选出最佳 Prompt 风格
5. **迭代优化** → 根据结果调整描述方式

**示例 Prompt 模板**：
```
[主体] + [动作] + [环境] + [光线] + [镜头] + [情绪]

例：一只橘猫在阳光下的窗台上伸懒腰，
    温暖自然光，特写镜头，慵懒舒适的氛围
```

**预期产出**：
- 3 个 5 秒 AI 生成视频片段
- 1 份 Prompt 效果对比笔记
- 初步掌握视频生成技巧
