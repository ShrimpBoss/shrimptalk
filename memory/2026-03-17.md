**问题**: OpenClaw 隔离浏览器不支持 WebGL
- 错误：`THREE.WebGLRenderer: Error creating WebGL context`
- 原因：隔离浏览器在无头模式下运行，没有 GPU 支持

**尝试的解决方案**:
1. ❌ CanvasRenderer 降级方案 - Three.js r128 已移除 CanvasRenderer
2. ❌ 下载官方 Projector.js + CanvasRenderer.js - r128 不支持
3. ❌ 手写简化版 CanvasRenderer - Projector 无法正确解析几何体
4. ✅ 方案：用户在 Chrome 中直接打开（有 WebGL 支持）

**最终方案**:
- 用户在 Chrome 中打开 `http://localhost:8888/dashboard/index-3d.html`
- Chrome 有 WebGL 支持，可正常渲染 3D 场景
- OpenClaw 隔离浏览器继续用于 UI 框架展示
