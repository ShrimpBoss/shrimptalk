**主题 2**: 营销心理学 (Marketing Psychology)

**具体内容**:
1. **用户决策双系统理论**
   - 系统 1 (快思考): 80% 购买决策由此驱动
   - 系统 2 (慢思考): 事后合理化

2. **核心心理效应**
   - 损失厌恶 (2:1 痛苦/快乐比例)
   - 锚定效应 (价格呈现顺序)
   - 选择悖论 (3 个选项最佳)

3. **定价策略**
   - 尾数定价 (¥99 vs ¥100)
   - 价格阶梯设计 (好/更好/最好)
   - 分解定价 ("每天只需 3 元")

4. **稀缺性/紧迫感设计**
   - 数量/时间/资格稀缺
   - 倒计时/库存条/"X 人正在看"
   - 使用原则：真实、具体、合理

5. **社会认同原理**
   - 6 种形式：专家/名人/用户/群众/朋友/认证
   - 评价优化策略
   - 社会证明元素清单

### 💡 核心收获（3-5 点）

1. **AI 视频已进入实用阶段**
   - Gen-4.5 画质足以满足大部分内容创作需求
   - 关键不是追求完美，而是快速迭代 + 批量生产
   - 对自媒体创作者是巨大机会窗口

2. **工具是放大器，创意是核心**
   - AI 降低制作门槛，但好内容仍需好创意
   - Prompt 工程是核心技能，需要持续积累
   - 建立个人 Prompt 库是当务之急

3. **决策是情感的，合理化是事后的**
   - 营销先打动情感，再提供理性支撑
   - 损失厌恶 > 获得欲望 (强调"错过失去"而非"得到")
   - 80% 决策由系统 1 驱动

4. **社会证明是最强转化杠杆**
   - 评价/用户数/媒体背书显著提升信任
   - 视频评价信任度是文字的 5 倍
   - 展示负面评价 + 专业回复显得真实

5. **稀缺性必须真实**
   - 虚假稀缺被发现会摧毁信任
   - "剩 3 件"比"库存紧张"更有效
   - 避免每天"最后一天"导致失效
