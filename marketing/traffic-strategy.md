## 数据追踪

### 核心指标
```
每日监控:
  - 花费（Spend）
  - 展示（Impressions）
  - 点击（Clicks）
  - CTR（点击率）
  - CPC（单次点击成本）
  - 转化（Conversions）
  - CPA（单次转化成本）
  - ROAS（广告支出回报率）
  - GMV（成交总额）
```

### 预警阈值
```
立即优化:
  - CTR < 1%：创意问题
  - CPC > $2：定向问题
  - 转化率 < 2%：落地页问题
  - ROAS < 2.0：关停或优化

表现优秀:
  - ROAS > 3.5：加预算 20-50%
  - ROAS > 5.0：加预算 100%+
```

## KOL 合作计划

### 筛选标准
```
微型 KOL（1 万 -10 万粉丝）:
  - 粉丝数：1 万 -10 万
  - 互动率：> 3%
  - 粉丝画像：匹配目标受众
  - 内容质量：高
  - 报价：$100-500/条

中型 KOL（10 万 -50 万粉丝）:
  - 粉丝数：10 万 -50 万
  - 互动率：> 2%
  - 报价：$500-2000/条
```

### 合作形式
```
1. 产品测评
   - 免费寄送产品
   - 真实使用体验
   - 附购买链接

2. 植入广告
   - 短视频植入
   - 图文提及
   - Stories 分享

3. 长期合作
   - 品牌大使
   - 月度内容
   - 专属折扣码
```
