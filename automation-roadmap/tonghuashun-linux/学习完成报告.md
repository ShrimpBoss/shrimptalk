### 基础操作流程

#### 流程 1：添加自选股
```
1. 搜索股票代码或名称
2. 按 Enter 进入个股页面
3. 按 Insert 键
4. 选择自选股分组
5. 确认添加
```

#### 流程 2：查看 K 线图
```
1. 进入个股页面
2. 按 F5 切换到 K 线图
3. 使用↓键选择周期
4. 使用↑↓键缩放图表
5. 使用←→键移动时间
```

#### 流程 3：查看技术指标
```
1. 在 K 线图界面
2. 输入指标代码（如 MACD）
3. 按 Enter 显示指标
4. 点击指标名称切换
```
