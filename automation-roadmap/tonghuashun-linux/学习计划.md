## 📝 常见问题

### Q1: 软件无法启动？
**解决**:
```bash
# 检查依赖
ldd /opt/apps/cn.com.10jqka/files/HevoNext.B2CApp

# 安装缺失依赖
sudo apt install libssl3 libicu70
```

### Q2: 如何添加自选股？
**方法**:
1. 搜索股票代码或名称
2. 按 Insert 键
3. 或点击"＋自选"按钮

### Q3: 如何切换 K 线周期？
**方法**:
1. 在 K 线图界面
2. 按↓键选择周期
3. 或使用菜单：周期→日线/周线/月线

### Q4: 如何查看技术指标？
**方法**:
1. 在 K 线图界面
2. 输入指标代码（如 MACD）
3. 或点击指标名称切换
