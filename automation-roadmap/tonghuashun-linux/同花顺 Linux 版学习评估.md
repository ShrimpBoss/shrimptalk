## 🔐 安全建议

### 账户安全

1. **使用独立密码**
   - 同花顺密码与其他网站不同
   - 定期更换密码

2. **启用双重验证**
   - 手机验证码
   - 动态口令

3. **资金密码分离**
   - 登录密码 ≠ 资金密码
   - 分别保管

### 设备安全

1. **使用虚拟机**
   - 完全隔离
   - 定期快照
   - 中毒可恢复

2. **专用环境**
   - 不用于其他用途
   - 不安装其他软件
   - 不浏览网页

3. **网络安全**
   - 使用 HTTPS
   - 不使用公共 WiFi
   - 使用 VPN（可选）

### 操作安全

1. **手动交易**
   - 所有交易必须手动确认
   - 不设置自动交易
   - 大额交易双重确认

2. **定期检查**
   - 检查持仓
   - 检查资金
   - 检查交易记录

3. **及时退出**
   - 使用后退出登录
   - 关闭浏览器/软件
   - 清除缓存
