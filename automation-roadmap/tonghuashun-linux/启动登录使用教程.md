## ⚠️ 当前问题

### libssl 版本不兼容

**错误信息**:
```
No usable version of libssl was found
Cannot get required symbol ERR_put_error from libssl
```

**原因**: 软件需要 libssl 1.1，但系统安装的是 libssl 3.0

**解决方案**:

### 方案 1：安装 libssl 1.1（推荐）
```bash
# 下载 libssl 1.1
wget http://archive.ubuntu.com/ubuntu/pool/main/o/openssl/libssl1.1_1.1.1f-1ubuntu2_amd64.deb

# 安装
sudo dpkg -i libssl1.1_1.1.1f-1ubuntu2_amd64.deb
```

### 方案 2：使用兼容模式
```bash
# 设置环境变量
export LD_LIBRARY_PATH=/opt/apps/cn.com.10jqka/files:$LD_LIBRARY_PATH

# 启动
/opt/apps/cn.com.10jqka/files/HevoNext.B2CApp
```

### 方案 3：使用 Wine 运行 Windows 版
如果 Linux 版无法运行，可以使用 Wine 运行 Windows 版本。

## 🎯 基础使用教程

### 任务 1：添加第一只自选股

**方法 1：搜索添加**
1. 在顶部搜索框输入股票代码（如 000001）
2. 按 Enter 键
3. 进入个股页面
4. 点击"＋自选"按钮
5. 选择自选股分组
6. 点击"确认"

**方法 2：快捷键添加**
1. 在搜索框输入股票代码
2. 按 Enter 进入个股
3. 按 Insert 键
4. 选择分组并确认

**示例**: 添加平安银行
```
搜索：000001 或 平安银行
操作：按 Insert 键
结果：添加到自选股
```

### 任务 2：查看股票行情

**查看大盘**:
```
按 F3 → 上证指数
按 F4 → 深证成指
```

**查看个股**:
```
1. 在自选股列表点击股票
2. 或搜索股票代码
3. 按 Enter 进入
```

**切换图表**:
```
按 F5 → K 线图/分时图切换
```

### 任务 3：查看 K 线图

**进入 K 线图**:
1. 进入个股页面
2. 按 F5 切换到 K 线图

**切换周期**:
```
↓ 键 → 选择周期菜单
选择：日线/周线/月线/分钟线
```

**缩放图表**:
```
↑ 键 → 放大
↓ 键 → 缩小
← → 键 → 左右移动
```

### 任务 4：查看技术指标

**添加指标**:
1. 在 K 线图界面
2. 输入指标代码
3. 按 Enter

**常用指标代码**:
```
MACD → 平滑异同移动平均线
KDJ  → 随机指标
BOLL → 布林线
MA   → 移动平均线
VOL  → 成交量
RSI  → 相对强弱指标
```

**切换指标**:
```
点击指标名称 → 切换不同指标
右键菜单 → 指标管理
```

### 任务 5：查看个股资料

**按 F10 查看**:
1. 进入个股页面
2. 按 F10 键
3. 查看公司资料

**资料内容**:
- 公司简介
- 财务数据
- 股东情况
- 重大事项
- 盈利预测

## ⚠️ 安全提醒

### 登录安全
- ✅ 使用强密码（8 位以上，字母 + 数字）
- ✅ 定期更换密码
- ✅ 不保存密码在软件中
- ✅ 使用后退出登录

### 交易安全
- ⚠️ F12 打开交易界面需格外小心
- ⚠️ 所有交易必须手动确认
- ⚠️ 不设置自动交易
- ⚠️ 大额交易双重确认

### 数据安全
- ✅ 行情数据只读（安全）
- ✅ 自选股本地保存
- ✅ 不上传敏感个人信息
- ✅ 定期备份自选股
