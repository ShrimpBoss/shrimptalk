## 📚 2026-03-20 交易体系学习 (20:00) — 周期框架深化

**学习主题**: 主升周期 vs 补涨周期、主升节点 vs 切换节点  
**学习时长**: 2 小时 (18:00-20:00)  
**输出文档**: memory/trading-study-2026-03-20.md (~9KB)

### 核心框架升级
- **原框架**: 五阶段（冰点→回暖→主升→分歧→退潮）
- **新框架**: 两周期 + 两节点（主升周期→补涨周期 + 主升节点→切换节点）

### 主升周期特征（5 个）
1. 连板高度不断突破（3 板→4 板→5 板→6 板→7 板）
2. 主线单一核心，资金聚焦
3. 梯队完整（从连板龙头到首板补涨）
4. 炸板率<25%（资金一致性强）
5. 成交量持续放大

### 补涨周期特征（5 个）
1. 连板高度受限（5-6 板天花板）
2. 主线分化（2-3 个主线）
3. 中位股断层（3 板、4 板空缺）
4. 炸板率 35%-45%（资金分歧加大）
5. 成交量震荡

### 主升节点判断（5 个信号）
1. 连板高度突破（5 板→6 板→7 板）
2. 炸板率下降（40%→30%→25%）
3. 主线聚焦（3 个→2 个→1 个）
4. 成交量放大（2 万亿→2.5 万亿→3 万亿）
5. 新题材出现

**操作**: 5 个信号出现 3 个以上 → **加仓到 7-10 成**

### 切换节点判断（5 个信号）
1. 连板高度无法突破（连续 3 天无法突破 6 板）
2. 炸板率持续上升（40%→45%→50%）
3. 旧主线退潮（主线龙头出现大面股）
4. 新题材试盘（有新题材但持续性不确定）
5. 成交量萎缩（2.3 万亿→2 万亿→1.8 万亿）

**操作**: 5 个信号出现 3 个以上 → **减仓到 3-5 成**

### 2026-03-20 市场定位
- **周期**: 补涨周期后期
- **节点**: 接近切换节点（但未完全确认）
- **仓位**: 5-7 成（防守为主）
- **策略**: 去弱留强，等切换节点确认后进攻

### 3. 情绪周期交易体系深化 — 两周期 + 两节点框架
- **原框架**: 五阶段（冰点→回暖→主升→分歧→退潮）
- **新框架**: 主升周期→补涨周期 + 主升节点→切换节点
- **主升周期特征**: 连板高度突破 (7 板+) / 单一主线 / 梯队完整 / 炸板率<25% / 成交量放大
- **补涨周期特征**: 连板高度受限 (5-6 板) / 主线分化 (2-3 个) / 中位股断层 / 炸板率 35%-45%
- **主升节点 5 信号**: 连板突破/炸板率下降/主线聚焦/成交量放大/新题材 (3 个确认→加仓 7-10 成)
- **切换节点 5 信号**: 高度无法突破/炸板率上升/旧主线退潮/新题材试盘/成交量萎缩 (3 个确认→减仓 3-5 成)
- **2026-03-20 定位**: 补涨周期后期，接近切换节点 (未完全确认)，仓位 5-7 成防守为主

### 4. 沟通表达双框架 — 清晰表达 + 非暴力沟通
- **金字塔原理**: 结论先行→理由→证据 (适合工作汇报)
- **PREP 法**: Point 观点→Reason 原因→Example 例子→Point 重申 (适合日常沟通)
- **倾听 5 层级**: 忽视→假装听→选择性听→专注听→同理心听 (最高级)
- **非暴力沟通 NVC 四步**: 观察 (不带评判)→感受 (真实情绪)→需求 (普世需要)→请求 (具体可行)
- **NVC 句式**: "当我看到 [观察]，我感到 [感受]，因为我需要 [需求]。你是否愿意 [请求]？"
- **沟通毒素**: 批评/蔑视/防御/石墙 (John Gottman 研究，蔑视最具破坏力)

### 5. 写作能力与故事结构 — 打动人心的文字
- **起承转合**: 起 (10-15% 引入)→承 (30-40% 发展)→转 (30-40% 高潮)→合 (15-20% 升华)
- **标题法则**: 数字法/对比法/疑问法/承诺法
- **AIDA 模型**: Attention 注意→Interest 兴趣→Desire 欲望→Action 行动
- **情感共鸣 4 杠杆**: 具体细节>抽象概念 / 感官描写>理性陈述 / 脆弱真实>完美无缺 / 普世情感>个人经历
- **Show Don't Tell**: "她盯着手机屏幕，直到自动锁屏，再点亮，再锁屏"vs"他很伤心"
