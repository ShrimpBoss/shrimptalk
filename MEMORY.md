### 2. 营销心理学核心框架
- 80% 购买决策由系统 1 (直觉) 驱动，系统 2 (理性) 只是事后合理化
- 三大武器：损失厌恶 > 锚定效应 > 稀缺性
- 定价三层：基础 (锚点) / 专业 (目标，标"最受欢迎") / 企业 (拉高锚点)

### 3. 情绪周期交易体系 — 宝哥核心方法论
- **六阶段循环**: 冰点→回暖→主升→分歧补涨→切换→退潮→冰点
- **仓位管理**: 冰点 0-1 成 / 回暖 2-3 成 / 主升 5-8 成 / 分歧 3-5 成 / 切换 2-3 成 / 退潮 0-1 成
- **核心认知**: 情绪周期不是精确科学，是概率框架；跟随不做对
- **关键指标**: 连板高度、涨停数量、炸板率、首板溢价、跌停数量
- **常见错误**: 冰点抄底、退潮接力、主升不上仓、恋战旧主线、分歧追高

### 4. 游资操盘手法系统化
- **龙头战法**: 只做市场最强的那只，辨识度 + 连板高度 + 成交量 + 题材逻辑
- **生命周期**: 启动期→确认期 (2-3 板最佳)→加速期→分歧期→衰退期
- **核心能力**: 判断市场情绪 (涨停数/炸板率/连板高度/龙头表现)
- **风控纪律**: 单股≤30% 仓位 / 浮盈加仓浮亏不加 / 止损坚决不扛单
- **龙虎榜价值**: 追踪游资席位行为，识别板块合力

### 5. 涨停板战法与情绪周期结合
- **涨停本质**: 资金态度最明确的表达，封板=一致看多，连板=人气共振
- **时间强度**: 一字板>秒板>早盘板>上午板>午后板>尾盘板
- **核心结论**: 情绪周期决定仓位和策略，涨停板战法是执行工具
- **打板条件**: 情绪回暖/主升 + 题材逻辑硬 + 辨识度高 + 量价配合 + 板块效应
- **首板/接力**: 首板适合主升期试错，接力只做龙头不做杂毛
