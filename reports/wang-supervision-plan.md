## 🚨 异常处理流程

### 级别 1: 轻微异常
**情况**: Agent 响应慢/输出延迟
**处理**: 
1. 提醒该 Agent
2. 记录延迟原因
3. 跟踪后续改进

### 级别 2: 中度异常
**情况**: 记忆保存未执行/输出质量差
**处理**:
1. 立即手动触发记忆保存
2. 要求重做低质量输出
3. 记录异常并上报李总

### 级别 3: 严重异常
**情况**: Agent 离线/编造数据/连续 2 次未保存
**处理**:
1. 立即唤醒/重启 Agent
2. 编造数据 → 零容忍，记录在案
3. 立即上报李总 + 提交整改方案
