## 🎯 优化方案

### 方案 A: 精简夜间任务

**保留核心**：
```
02:00 🤖 夜间自动修复（核心）
03:00 📚 夜间知识整理（核心）
```

**删除**：
- ❌ 01:00 系统自检（与自动修复重复）

3. **题材强度分析** - 题材集中度判断
