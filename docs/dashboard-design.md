## 🛠️ 技术选型

### 方案 A: Three.js（推荐）
- **优点**: 成熟、文档全、社区大
- **3D 能力**: 完整 3D 场景
- **动画**: 支持骨骼动画
- **难度**: 中等

### 方案 B: Spline
- **优点**: 设计友好、可导出代码
- **3D 能力**: 中等
- **动画**: 预设动画多
- **难度**: 简单

### 方案 C: Live2D / Pixi.js
- **优点**: 轻量、2.5D 效果好
- **3D 能力**: 有限
- **动画**: 2D 骨骼
- **难度**: 简单

### 方案 D: Canvas + CSS 动画（MVP 版本）
- **优点**: 最简单、快速上线
- **3D 能力**: 无（2D 卡通）
- **动画**: CSS 动画
- **难度**: 非常简单

## 🚨 阻塞检测逻辑

```javascript
// 伪代码
function checkBlocked() {
  const currentTask = getCurrentTask();
  const expectedDuration = getExpectedDuration(currentTask);
  const actualDuration = Date.now() - currentTask.startedAt;
  
  if (actualDuration > expectedDuration * 2) {
    triggerAlert({
      type: 'blocked',
      task: currentTask.name,
      duration: actualDuration,
      message: '虾老板好像卡住了，要救他出来吗？'
    });
  }
}
```

## 💾 记忆保存机制

### 自动保存
- 每 5 分钟自动保存当前状态
- 保存到 `memory/dashboard-state.json`
- Git 提交历史记录

### 救援时保存
```bash
# 救援脚本
1. 保存当前任务状态
2. 保存所有 Agent 状态
3. 保存未提交的记忆
4. Git commit
5. 重启/重置任务
```
