# 🤖 飞书自定义机器人配置指南

## 📋 配置步骤（只需 3 分钟）

### 步骤 1：在飞书群组中添加机器人

1. **进入目标群组** → 点击右上角「设置」图标
2. **群机器人** → 点击「添加机器人」
3. **选择「自定义机器人」**
4. **设置机器人信息**：
   - 名称：Dashboard 助手
   - 头像：可选
   - 描述：自动推送 Dashboard 通知
5. **点击「添加」**

### 步骤 2：获取 Webhook 地址

添加成功后，你会看到：
```
https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxx
```

**⚠️ 重要**：
- 复制并保存这个 webhook 地址
- 不要泄露给他人
- 不要提交到 Git

### 步骤 3：配置安全设置（强烈推荐）

#### 方式一：签名校验（最安全）⭐

1. 点击机器人头像进入配置
2. 选择「签名校验」
3. 复制 **Secret**（类似：`SECxxxxxxxxxx`）
4. 点击「保存」

#### 方式二：IP 白名单

1. 选择「IP 白名单」
2. 添加你的服务器 IP
3. 点击「保存」

#### 方式三：自定义关键词

1. 选择「自定义关键词」
2. 添加关键词：`Dashboard`、`报警`、`通知`
3. 点击「保存」

### 方法 2：使用 Python 脚本

创建测试脚本 `test_feishu.py`：

```python
#!/usr/bin/env python3
import hashlib
import hmac
import base64
import time
import json
import requests
from dotenv import load_dotenv
import os

# 加载环境变量
load_dotenv('.env')

WEBHOOK_URL = os.getenv('FEISHU_WEBHOOK_URL')
SECRET = os.getenv('FEISHU_SECRET')

def gen_sign(timestamp, secret):
    """生成签名"""
    string_to_sign = f'{timestamp}\n{secret}'
    hmac_code = hmac.new(
        string_to_sign.encode('utf-8'),
        digestmod=hashlib.sha256
    ).digest()
    sign = base64.b64encode(hmac_code).decode('utf-8')
    return sign

def send_message(text, sign=False):
    """发送消息到飞书"""
    timestamp = str(int(time.time()))
    
    data = {
        'msg_type': 'text',
        'content': {
            'text': text
        }
    }
    
    if sign and SECRET:
        data['timestamp'] = timestamp
        data['sign'] = gen_sign(timestamp, SECRET)
    
    response = requests.post(WEBHOOK_URL, json=data)
    result = response.json()
    
    if result.get('code') == 0 or result.get('StatusCode') == 0:
        print('✅ 发送成功！')
        return True
    else:
        print(f'❌ 发送失败：{result}')
        return False

if __name__ == '__main__':
    # 测试发送
    send_message('🤖 Dashboard 测试消息', sign=True)
```

安装依赖：
```bash
pip install python-dotenv requests
```

运行测试：
```bash
cd ~/.openclaw/workspace/feishu-bot
python3 test_feishu.py
```

## 🚀 集成到 Dashboard

### 方案 1：后端 Node.js 集成

在 `dashboard-website/backend/server.js` 添加：

```javascript
// 飞书通知模块
const crypto = require('crypto');

function genFeishuSign(timestamp, secret) {
  const stringToSign = `${timestamp}\n${secret}`;
  const hmac = crypto.createHmac('sha256', stringToSign);
  hmac.update('');
  const sign = hmac.digest('base64');
  return sign;
}

async function sendFeishuNotification(message, options = {}) {
  const webhookUrl = process.env.FEISHU_WEBHOOK_URL;
  const secret = process.env.FEISHU_SECRET;
  
  const timestamp = String(Math.floor(Date.now() / 1000));
  
  const data = {
    msg_type: options.msgType || 'text',
    content: options.content || { text: message }
  };
  
  if (secret) {
    data.timestamp = timestamp;
    data.sign = genFeishuSign(timestamp, secret);
  }
  
  try {
    const response = await fetch(webhookUrl, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data)
    });
    
    const result = await response.json();
    return result.code === 0 || result.StatusCode === 0;
  } catch (error) {
    console.error('飞书通知失败:', error);
    return false;
  }
}

// 使用示例：建议完成时发送通知
app.put('/api/suggestions/:id', authenticateToken, async (req, res) => {
  // ... 更新建议逻辑 ...
  
  // 发送飞书通知
  await sendFeishuNotification(
    `✅ 建议已完成\n\n标题：${suggestion.title}\n负责人：${suggestion.assignee}\n反馈：${suggestion.feedback}`,
    {
      msgType: 'post',
      content: {
        post: {
          zh_cn: {
            title: '建议完成通知',
            content: [[
              { tag: 'text', text: '建议已完成：' },
              { tag: 'text', text: suggestion.title },
              { tag: 'at', user_id: 'all' }
            ]]
          }
        }
      }
    }
  );
  
  res.json({ success: true });
});
```

## ⚠️ 注意事项

### 频率限制
- **100 次/分钟** 单租户单机器人
- **5 次/秒** 瞬时并发
- **避开整点/半点**（10:00、17:30 等高峰时段）

### 消息大小
- 请求体不超过 **20 KB**

### 时间戳
- 签名校验时，时间戳必须在 **1 小时内**
- 校准服务器时间

### 错误码
| Code | 说明 | 解决方案 |
|------|------|----------|
| 0 | 成功 | - |
| 19021 | 签名失败 | 检查时间戳和密钥 |
| 19022 | IP 不在白名单 | 添加 IP 到白名单 |
| 19024 | 关键词不匹配 | 消息包含至少一个关键词 |
| 9499 | 请求体格式错误 | 检查 JSON 格式 |

## 🔧 故障排查

### 1. 消息发送失败

**检查**：
- Webhook 地址是否正确
- 网络连接是否正常
- JSON 格式是否正确

**测试**：
```bash
curl -v -X POST -H "Content-Type: application/json" \
  -d '{"msg_type":"text","content":{"text":"test"}}' \
  https://open.feishu.cn/open-apis/bot/v2/hook/YOUR_WEBHOOK
```

### 2. 签名校验失败

**检查**：
- 时间戳是否超过 1 小时
- 服务器时间是否准确
- 签名算法是否正确

**校准时间**：
```bash
# Linux
sudo ntpdate -s time.nist.gov

# macOS
sudo sntp -sS time.apple.com
```

### 3. 关键词校验失败

**检查**：
- 消息内容是否包含至少一个关键词
- 关键词是否只匹配文本内容（不匹配链接）
