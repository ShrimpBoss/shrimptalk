## 🎯 你只需要做 3 件事

### 1️⃣ 在飞书中添加机器人（2 分钟）

1. 打开飞书，进入你想接收通知的群组
2. 点击右上角「设置」→「群机器人」
3. 点击「添加机器人」→ 选择「自定义机器人」
4. 设置名称：`Dashboard 助手`
5. 点击「添加」
6. **复制 Webhook 地址**（重要！）
7. （推荐）配置「签名校验」，复制 Secret

### 2️⃣ 运行配置脚本（1 分钟）

```bash
cd ~/.openclaw/workspace/feishu-bot
./setup-feishu.sh
```

脚本会提示你：
- 粘贴 Webhook 地址
- 是否启用签名校验（推荐选 1）
- 粘贴 Secret（如果启用签名）

### 3️⃣ 测试发送

```bash
cd ~/.openclaw/workspace/feishu-bot
python3 test_feishu.py
```

如果看到 `✅ 发送成功！`，说明配置完成！

## 🔐 安全说明

### 你需要保护的信息

1. **Webhook URL** - 机器人的调用地址
2. **Secret** - 签名密钥（如果启用）

### 已做的安全措施

✅ `.env` 文件权限设置为 600（仅所有者可读写）
✅ 配置文件不会提交到 Git
✅ 使用签名校验防止恶意调用

### 你还需要做的

⚠️ 不要在公开场合分享 Webhook URL
⚠️ 不要将 `.env` 文件上传到 GitHub/GitLab
⚠️ 定期更换 Secret（建议每 3-6 个月）

## ⚠️ 常见问题

### Q1: 消息发送失败？
**检查**：
- Webhook 地址是否正确
- 网络连接是否正常
- 是否超过频率限制（100 次/分钟）

### Q2: 签名校验失败？
**检查**：
- Secret 是否正确复制
- 服务器时间是否准确
- 时间戳是否超过 1 小时

### Q3: 如何关闭通知？
**方法**：
- 在飞书群组中移除机器人
- 或在 Dashboard 配置中禁用飞书通知
