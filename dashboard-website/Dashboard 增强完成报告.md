### 2. 后端 API 增强 ✅

#### Agent 状态 API
```javascript
GET /api/agents/status

返回 10 人完整团队：
- 新增字段：role（角色）
- 新增字段：efficiency（效率）
- 新增字段：pending（待处理）
- 新增字段：rating（好评率）
```

#### 建议管理 API
```javascript
POST   /api/suggestions              - 提交建议
PUT    /api/suggestions/:id          - 更新建议
DELETE /api/suggestions/:id          - 删除建议
POST   /api/suggestions/batch-update - 批量更新
```

## 📝 API 使用示例

### 获取 Agent 状态
```bash
curl http://localhost:3000/api/agents/status
```

**返回示例**:
```json
{
  "success": true,
  "data": {
    "agents": [
      {
        "name": "王总",
        "role": "副经理",
        "status": "running",
        "task": "待审核：3 项 · 预警：0"
      },
      {
        "name": "小服",
        "role": "客服",
        "status": "running",
        "task": "待回复：12 条 · 好评率：98%"
      },
      ...
    ]
  }
}
```

### 提交建议
```bash
curl -X POST http://localhost:3000/api/suggestions \
  -H "Content-Type: application/json" \
  -d '{
    "title": "优化 Dashboard 加载速度",
    "content": "建议优化前端资源加载",
    "priority": "high",
    "category": "系统功能",
    "assignee": "小安"
  }'
```
