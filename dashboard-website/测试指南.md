## 📝 完成反馈示例

如果你想测试完整的「提交→处理→反馈」流程，可以手动更新数据库：

```bash
cd /home/terrence/.openclaw/workspace/dashboard-website/backend
node -e "
const Database = require('better-sqlite3');
const db = new Database('./dashboard.db');

// 更新某条建议为已完成并添加反馈
db.prepare(\`
  UPDATE suggestions 
  SET status = 'completed', 
      progress = 100, 
      feedback = '✅ 已完成！\n\n优化措施：\n1. 压缩图片资源（减少 60% 体积）\n2. 启用 CDN 加速\n3. 实现懒加载\n\n结果：加载时间从 3s 降至 0.8s ⚡',
      completed_at = CURRENT_TIMESTAMP
  WHERE id = 4
\`).run();

console.log('✅ 建议已更新为已完成状态');
"
```

然后刷新 Dashboard，会看到第 4 条建议显示绿色反馈框。

## ⚠️ 常见问题

### 1. 页面打不开
```bash
# 检查前端服务器
ps aux | grep "http.server"

# 如果没运行，启动它
cd /home/terrence/.openclaw/workspace/dashboard-website/frontend
python3 -m http.server 8888 &
```

### 2. 数据不显示
```bash
# 检查后端服务器
ps aux | grep "node server.js"

# 查看后端日志
tail -f /tmp/backend.log 2>/dev/null || echo "日志文件不存在"
```

### 3. API 返回错误
```bash
# 测试 API 是否正常

# 如果返回 403，说明认证有问题，检查后端代码
```
