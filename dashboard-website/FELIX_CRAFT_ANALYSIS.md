## 核心改进点

### 1. 深色主题（Dark Mode）

**Felix Craft 风格：**
- 背景：纯黑 `#0A0A0A` 或深灰 `#111111`
- 卡片：深灰 `#1A1A1A`
- 文字：白色 `#FFFFFF` + 灰色 `#888888`
- 强调色：金色 `#F5C542` 或品牌色

**我们的改进：**
```css
:root {
  --bg-primary: #0A0A0A;
  --bg-card: #141414;
  --text-primary: #FFFFFF;
  --text-secondary: #A0A0A0;
  --accent: #6B2CF5;  /* 保持品牌紫色 */
  --success: #00C48C;
  --danger: #FF4949;
}
```

---

### 2. 简化指标卡片

**Felix Craft 风格：**
```
┌─────────────────┐
│ REVENUE (7D)    │  ← 小标题，灰色
│ $42,080.40      │  ← 大数字，金色
└─────────────────┘
```

**我们当前：**
```
┌─────────────────┐
│ 💰              │  ← emoji 图标
│ $1,234          │
│ GMV             │
│ ↑ 12.5%         │
└─────────────────┘
```

**改进方案：**
- 去掉 emoji（更专业）
- 标题缩小 + 灰色
- 数字加大 + 加粗
- 趋势用颜色标识（绿涨红跌）

### 4. 数据可验证性

**Felix Craft 每个数据都有验证链接：**
- "Verify on Trueskill"
- "View on BaseScan"
- "View on Clanker"

**我们可以添加：**
- GMV → "查看 Shopify 后台"
- 订单 → "查看订单列表"
- 广告花费 → "查看 Meta 广告后台"
- 产品成本 → "查看 CJ 采购链接"

---

### 5. 图表优化

**Felix Craft 的 50 天趋势图：**
- 简洁网格背景
- 清晰的图例
- 多系列对比（4 个业务线）
- 无多余装饰

**我们的 7 天图改进：**
- 增加 30 天/90 天切换
- 添加网格线
- 支持多系列对比（GMV + 订单 + 利润）
- 可点击图例隐藏/显示系列

---

### 6. 实时数据标识

**Felix Craft：**
```
Updated 3/8/2026, 3:22:21 PM
```

**我们改进：**
- 显示精确到分钟的最后更新时间
- 超过 30 分钟未更新显示黄色警告
- 超过 1 小时显示红色警告

## 设计原则

1. **Less is More** - 只展示关键数据
2. **Data First** - 数字清晰可读
3. **Verifiable** - 每个数据可追溯
4. **Real-time** - 实时更新
5. **Professional** - 深色主题 + 简洁设计
